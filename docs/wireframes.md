# GaitKnee-View 画面モック（ワイヤーフレーム）

## 概要
MediaPipeを使用した膝関節 lateral thrust 可視化Webアプリのワイヤーフレーム。
モバイル・デスクトップ両対応のレスポンシブデザイン。

---

## モバイル画面（3画面構成）

![モバイルワイヤーフレーム](/Users/kobayashiryuju/ryuju-personal/アプリ生成/GaitKnee-View/docs/wireframes_mobile.png)

### ① ホーム画面
| 要素 | 詳細 |
|------|------|
| ヘッダー | アプリ名「GaitKnee-View」 |
| メインCTA | 「新規撮影」大きなボタン（カメラアイコン付き） |
| セカンダリ | 「過去データ」ボタン |
| ビジュアル | 歩行シルエット図（膝にハイライト） |

### ② 撮影画面
| 要素 | 詳細 |
|------|------|
| プレビュー | 全画面カメラプレビュー |
| ガイド | 人体シルエット枠（位置決め用） |
| 録画ボタン | 大きな赤丸ボタン |
| タイマー | 録画時間表示「00:00」 |
| カメラ切替 | 右上にフリップアイコン |

### ③ 結果画面
| 要素 | 詳細 |
|------|------|
| スケルトン | スティックフィギュアの再生プレビュー |
| 軌跡 | 膝の2D軌跡（赤でthrust部分をハイライト） |
| 波形グラフ | 左右別のlateral thrust波形 |
| サマリー | 「右膝: 強」等のバッジ表示 |
| アクション | 「保存」「やり直し」ボタン |

### レイアウト構成

```
┌─────────────────────────────────────────────────────────┐
│  GaitKnee-View                               [保存]     │
├─────────────────────┬───────────────────────────────────┤
│                     │   膝軌跡（フロント面）              │
│  スケルトン          │   ┌───────────────────┐          │
│  アニメーション       │   │  X-Y 軌跡プロット   │          │
│                     │   │  （赤でthrust表示）  │          │
│  [▶ ━━━━━━━━━━]     │   └───────────────────┘          │
├─────────────────────┼───────────────────────────────────┤
│                     │   右膝 Lateral Thrust             │
│                     │   ～～～∧～～～～∧～～～          │
│      サマリー        │   左膝 Lateral Thrust             │
│      カード          │   ～～～～～～～～～～～          │
│                     │                                   │
│                     │   [振幅] [左右差] [評価]           │
└─────────────────────┴───────────────────────────────────┘
```

### パネル構成

| パネル | 幅 | 内容 |
|--------|-----|------|
| 左パネル | 40% | スケルトンアニメーション + 再生コントロール |
| 右上パネル | 60% | 膝軌跡の2Dプロット（フロント面） |
| 右下パネル | 60% | 左右別の波形グラフ + サマリーカード |
